## 三、技术路线

### 1. 后端技术栈
- **框架**：FastAPI
- **数据库**：SQLAlchemy + SQLite/MySQL
- **大语言模型**：DeepSeek API
- **语音识别**：Whisper
- **音频处理**：FFmpeg

### 2. 前端技术栈
- **框架**：Vue.js 3 (组合式API)
- **UI**：自定义CSS + 响应式设计
- **状态管理**：Vue 3 Composition API
- **通信**：WebSocket

### 3. 数据流设计
1. 前端采集音频 → WebSocket → 后端
2. 后端处理音频 → Whisper识别 → 文本结果
3. 文本结果 → DeepSeek API分析 → 结构化数据
4. 结构化数据 → 数据库存储
5. 前端获取结果 → 实时展示
